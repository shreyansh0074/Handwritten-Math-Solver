**# Handwritten Mathematical Expression Solver**



**An end-to-end deep learning based OCR system that recognizes handwritten digits and operators and solves mathematical expressions.**



**## Features**

**- Recognizes handwritten digits (0–9) and operators (+, −, ×, ÷)**

**- Uses CNN (PyTorch) for symbol classification**

**- OpenCV-based image segmentation \& preprocessing**

**- Streamlit web app for real-time image upload and prediction**



**## Tech Stack**

**- Python, PyTorch**

**- OpenCV, NumPy**

**- Streamlit**



**## How to Run Locally**



**```bash**

**pip install -r requirements.txt**

**streamlit run app.py**
